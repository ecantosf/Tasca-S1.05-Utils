FILE MANAGEMENT TEST UTILITY
=============================

Description:
A simple Java program to demonstrate basic file operations including:
- Creating directories
- Creating text files
- Reading file content
- Writing to files
- Listing directory contents
- Copying and moving files
- Deleting files and directories

Project Structure:
src/
  Main.java           - Main program entry point
  FileManager.java    - Core file operations
  examples/           - Example usage files
test-files/           - Test directory for file operations

Usage:
1. Compile: javac src/Main.java
2. Run: java src.Main
3. Follow on-screen menu options

Features Tested:
✓ Directory creation and navigation
✓ File I/O operations
✓ Exception handling for file operations
✓ Resource management with try-with-resources

Created for: Java File Management Exercise
